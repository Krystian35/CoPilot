# Lab 10 ‚Äî Generate a Full CRUD Python Web Application using Copilot

### üéØ Goal

Use **GitHub Copilot** to generate a complete web application using:

* **Python + Flask**
* **Jinja2 templates**
* **Bootstrap UI**
* **SQLite database**

The application must support CRUD operations for two tables:

* `categories`
* `products`

Database structure and sample data are provided in the `schema.sql` file.
The goal is to let **Copilot generate the application**, not manually write the code.

---

## ‚úÖ What we will do

1. Create a new project folder in VS Code.
2. Add the provided `schema.sql` to the project.
3. Use Copilot Chat to generate the app structure and code.
4. Copilot will create:

   * folders (`app/`, `database/`)
   * Flask application setup
   * CRUD routes for Products and Categories
   * Jinja templates using Bootstrap
   * requirements.txt, README.md, MIT License
5. Run the application and verify CRUD operations.
6. Commit and push to GitHub.

> üß† You are not coding ‚Äî you are guiding Copilot.

